Run it locally

# from ~/clovion-cms
pnpm install && pnpm db:up
pnpm prisma:migrate && pnpm db:seed
pnpm dev # → http://localhost:3007
pnpm worker # scheduled-publish + webinar jobs (2nd terminal)

# login: admin@clovion.ai / ChangeMe123!

To switch on live AI

Settings → AI → paste an OpenRouter API key, choose a default + embedding model, Test connection. Then AI Write in the editor streams real drafts grounded in your active SOP + Knowledge Base. Without a key, the engine returns a graceful "not configured" message — everything else runs fully.
